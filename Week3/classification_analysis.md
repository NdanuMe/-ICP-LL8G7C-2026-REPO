# Classification Model Analysis

## Objective
To predict whether a passenger survived using the Titanic dataset.

## Models Used
1. Logistic Regression
2. Decision Tree

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- Confusion Matrix

## Results
Logistic Regression performed well for binary classification.

Decision Tree provided an alternative model but may overfit.

## Conclusion
Logistic Regression was chosen as the final model due to its stability and interpretability.

## Key Learning
- Classification models predict categorical outcomes
- Feature selection improves model performance
- Confusion matrix helps understand prediction errors
