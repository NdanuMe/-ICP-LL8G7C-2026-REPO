# Regression Model Analysis

## Objective
The goal of this project was to predict house prices using machine learning models.

## Models Used
1. Linear Regression
2. Random Forest Regressor

## Evaluation Metrics
- RMSE (Root Mean Squared Error)
- MAE (Mean Absolute Error)

## Results
Linear Regression provided a baseline model for prediction.

Random Forest performed better by capturing complex patterns in the data, resulting in lower RMSE.

## Conclusion
Random Forest is more accurate for this dataset, while Linear Regression is simpler and faster.

## Key Learning
- Ensemble models improve prediction accuracy
- Evaluation metrics are important for comparing models
- Data preprocessing impacts model performance
